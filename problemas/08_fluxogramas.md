# Fluxogramas

1 - Desenhe o fluxograma do seguinte programa:

```c
#include <stdio.h>
int main() {
    unsigned int i;
    for (i = 8; i < 16; ++i) {
        printf("%x\n", i);
    }
    return 0;
}
```
